# py-popsift
this is a python binding of [popsift](https://github.com/alicevision/popsift)(a open-source implementation of the sift algorithm in CUDA) , you can use it in python-opencv .
It usually speed up 15X then cpu . 
## test environment
* cuda 10.2 
* gcc 8.3.1
* python 3.7.6
* pybind11 2.6.1 
* opencv 4.5.0
